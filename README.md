# Tasked

### From _**MeshaMakes**_

[![Generic badge](https://img.shields.io/badge/status-In%20Progress-yellowgreen)]()

Tasked is a simple todo list app made using flutter, with a simple yet effective UI Interface that anyone can get used to.

## Features

- Adding a todo 🗒️
- Removing a todo ✖️
- Making complete ✔️
- Free 🆓
- Responsive Layout 💻
